# Chokri Language — Linguistics Reference
*Source: Tetseo & Gope (2024), JJLL 7(2):38–70*

## Overview
- **Family**: Sino-Tibetan > Tibeto-Burman
- **ISO 639-3**: nri
- **Location**: Phek District, Nagaland, India
- **Typology**: Agglutinative, SOV word order, postpositional
- **Tones**: 5 lexical tones (extra-high, high, mid, low, mid-rising)
- **Script**: Latin-based practical orthography (used in Bible corpus); IPA notation used in academic papers

## Word Order
- **SOV** (Subject–Object–Verb)
- Adjectives **follow** nouns (e.g., "shirt black" not "black shirt")
- Postpositions **follow** nouns (unlike English prepositions)

## Pronouns
| Person | Singular | Dual (incl.) | Dual (excl.) | Plural (incl.) | Plural (excl.) |
|--------|----------|--------------|--------------|----------------|----------------|
| 1st | í | ámì / āwú | hā-ní | ú-kɔ̄ | hā-kɔ̄ |
| 2nd | nɔ̄ | nɛ̄-ní | — | nɛ̄-kɔ̄ | — |
| 3rd | ū / pū | ū-ní / pū-ní | — | ū-kɔ̄ | — |

- **Gender-neutral** third person (no he/she distinction)
- Distinguishes **inclusive vs exclusive** first person plural

## Demonstrative Pronouns
| Form | Meaning |
|------|---------|
| hīhī | this (near speaker) |
| tsə̀tsə̀ | that (within vision) |
| lűhī | that (within vision, further) |
| sə̄sə̄ | that (outside vision) |

Number is marked as **infix**: hīhī → hī-ní-hī (these two) → hī-kɔ̄-hī (these)

## Key Verbal Affixes
| Affix | Function | Example |
|-------|----------|---------|
| -vá | Progressive | tì-vá = eating |
| -lù | Continuative | tì-lù = still eating |
| -bɔ̄ | Inceptive | tì-bɔ́ = started eating |
| -zɔ̄ / -sɘ̋ | Iterative | tì-zɔ̄ = kept eating |
| -vɛ̋ | Perfective | tì-vɛ̋ = have eaten |
| -tɛ́ | Perfect / Prospective | tì-tɛ́ = have eaten (non-finite) |
| -jɔ̄ | Habitual / Gnomic | tì-jɔ̄ = eat (habitually) |
| -jì / kɘ̄jì | Resultative | pāssə̀-kɘ̄jì = having passed |
| -tɔ̀ | Future tense | tì-tɔ̀ = will eat |
| -tɔ̌-zə́ | Prospective (future) | tì-tɔ̌-zə́ = going to eat |

## Number Marking (nouns)
- Singular: unmarked
- Dual: suffix -ní (e.g., lɛ̄sı̋dà-ní = two books)
- Plural: suffix -kɔ̄ (e.g., lɛ̄sı̋dà-kɔ̄ = books)
- Not obligatory when numerals present

## Derivational Prefixes
| Prefix | Function | Example |
|--------|----------|---------|
| kɘ̄- / ɛ̄- | Nominalizer (verb/adj → noun) | mɘ̄lɘ̄ (believe) → kɘ̄-mɘ̄lɘ̄ (belief) |
| tʰɛ̄- | Nominalizer (verb → noun) | dʒɔ́ (bless) → tʰɛ̄-dʒɔ̋ (blessing) |
| mɛ̄- | Causative | tì (eat) → mɛ̄-tì (feed/cause to eat) |
| -mì | Agent nominalizer | kɘ̄-mɘ̄tʰa̋-mì = teacher |

## Orthography Note
The Bible corpus uses a **practical orthography** (Latin script without tone marks).
Academic papers use **IPA with diacritics** marking 5 tones.
The 115 pairs from Tetseo & Gope (2024) use IPA notation — this is intentional, adding diversity.
